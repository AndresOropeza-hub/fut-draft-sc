# Fut Draft Manager 5v5 - SC ⚽
Aplicación web para gestionar sorteos de fútbol 5 estilo FIFA Draft, con persistencia en la nube y generación de reportes visuales.

## Características:
- 🏃‍♂️ **Gestión de 15 jugadores** con estadísticas de ATK/DEF.
- 🧤 **Fase de Porteros:** Selección prioritaria de guardametas.
- 🎖️ **Fase de Capitanes:** Basada en el rendimiento (OVR).
- 🎨 **Personalización:** Selección de uniformes con efectos visuales (Auras).
- 📥 **Exportación:** Descarga de resumen final optimizado para compartir.
- ☁️ **Persistencia:** Sincronización automática con JSONBin.
